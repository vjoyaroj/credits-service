# credits-service

## Descripcion
Microservicio de gestion de creditos.

## Endpoints
- `GET /api/v1/credits`
- `POST /api/v1/credits`
- `GET /api/v1/credits/{id}`
- `PUT /api/v1/credits/{id}`
- `DELETE /api/v1/credits/{id}`
